# Index format

## `INDEX.md` (tracked, human-facing)

Regenerated by `./bin/memory-bridge index rebuild` (full rewrite).

- **Header**: `Last rebuilt`, `Total conversations`, `Conversations with open items` (IST timestamp).
- **Summary table** columns: `#`, `Date`, `Agent`, `Topics`, `Keywords`, `Open`, `File`.
- **`## Entries`**: per-conversation blocks with file path, topics, keywords, open count, related sessions, and a short **Context** excerpt (first lines under `## Context`).

Do not edit `INDEX.md` by hand.

## `.index/` (local, gitignored)

Sidecars for tooling and freshness without re-parsing markdown on every command:

- `.index/manifest.json` — `rebuilt_at_ist`, counts, `content_sha256_by_rel_path` map.
- `.index/conversations/*.json` — one JSON per conversation (`rel_path`, frontmatter echoes, open texts, excerpts, warnings).

`.index/` is **not** part of the cross-agent Git contract unless you deliberately stop ignoring it; keep Git traffic small and treat sidecars as build artifacts.

## Freshness checks

- **Agent / skill**: Compare `conversations/**/*.md` count vs `Total conversations` in `INDEX.md`.
- **CLI**: `./bin/memory-bridge index check` (uses manifest hashes when `.index/` exists).
