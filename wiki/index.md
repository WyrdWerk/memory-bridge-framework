# Memory Bridge Wiki

Compiled knowledge articles synthesized from conversations. Generated by the `llm-wiki-compile` skill.

## Topics

No compiled topics yet. The wiki starts empty — you populate it by running `llm-wiki-compile` after saving conversations.

## Getting Started

1. Save conversations to `conversations/YYYY/MM/DD/`
2. Add topic definition to `.registry.yaml`
3. Create `raw/{topic}/` directory for source material
4. Create `wiki/{topic}/` directory for compiled articles
5. Run `./bin/memory-bridge wiki-compile {topic}` or `/llm-wiki-compile {topic}`

See [WIKI_TEMPLATE.md](WIKI_TEMPLATE.md) for the article format and workflow.
